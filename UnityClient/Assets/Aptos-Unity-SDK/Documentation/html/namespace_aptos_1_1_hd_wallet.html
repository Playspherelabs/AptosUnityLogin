enum Aptos.HdWallet.SeedMode

Specifies the available seed modes for key generation.

Available modes:

SeedMode.Ed25519Bip32

SeedMode.Bip39

Enumerator
Ed25519Bip32 

Generates Ed25519 based BIP32 keys.

This seed mode is compatible with the keys generated in the Sollet/SPL Token Wallet, it does not use a passphrase to harden the mnemonic seed.

Bip39 

Generates BIP39 keys.

This seed mode is compatible with the keys generated in the solana-keygen cli, it uses a passphrase to harden the mnemonic seed.
